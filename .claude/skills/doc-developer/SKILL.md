# Developer Documentation Skill

This skill generates comprehensive technical documentation for software engineers.

## Instructions

When generating developer documentation, follow the guidelines in SKILL_DOC_DEVELOPER.md.

### Documentation Process
1. Analyze the codebase structure and architecture
2. Identify key technical components and patterns
3. Document APIs, modules, and data flows
4. Include setup instructions and technical details
5. Generate DOCUMENTATION_DEVELOPER.md

### Key Principles
- Focus on HOW things work technically
- Include code examples and API references
- Document architecture decisions and patterns
- Provide setup and deployment instructions
- Keep technical depth high

## Supporting Files
The SKILL_DOC_DEVELOPER.md file contains the documentation standards and templates.
